The What's Up WordPress? plugin exposes various bits of data to the What's Up WordPress desktop application.

== Description ==

The What's Up WordPress? plugin simply exposes some needed data to [the associated desktop application](http://www.jamesdimick.com/creations/whats-up-wordpress/). It's secure because it requires valid WordPress login credentials before the data is released.

**Get more support at [the official What's Up WordPress project page](http://www.jamesdimick.com/creations/whats-up-wordpress/).**

== Installation ==

1. Extract all files from the ZIP archive, making sure to keep the file structure intact.
2. Upload the `whats-up-wordpress` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin through the `Plugins` menu in WordPress.
5. Grab the [What's Up WordPress? desktop application](http://www.jamesdimick.com/creations/whats-up-wordpress/) and add your site.

**See Also:** ["Installing Plugins" article on the WP Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
